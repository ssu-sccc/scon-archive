불꽃놀이의 아름다움
time limit per test
3 seconds
memory limit per test
1024 megabytes
input
standard input
output
standard output

트리 $$$G(V, E)$$$가 주어진다. 찬솔이가 서 있을 정점 $$$u$$$를 하나 고른 뒤 폭죽을 터뜨린다. $$$u$$$를 제외한 모든 정점 $$$v\in V$$$에서 폭죽이 터지며, 정점 $$$u$$$에서 정점 $$$v$$$까지의 거리는 $$$D_v$$$이다. 정점 $$$v$$$에서 터지는 폭죽의 아름다움은 $$$W_v \times D_v$$$이다. 불꽃놀이의 아름다움은 모든 정점 $$$v\in V (u\neq v)$$$에서 터지는 폭죽의 아름다움($$$W_v \times D_v$$$)의 합으로 정의된다. 즉, $$$\sum_{v\in V, u\neq v} W_v \times D_v$$$ 이다. 찬솔이가 서 있을 정점을 잘 조정하여 불꽃놀이의 아름다움의 최댓값을 구해주자.

Input

첫째 줄에 정점의 수 $$$N$$$이 주어진다. $$$(2\leq N\leq 200\,000)$$$ 둘째 줄부터 $$$N-1$$$개의 줄에 걸쳐 간선을 의미하는 정수 $$$u\ v$$$가 공백으로 구분되어 주어진다. $$$u$$$와 $$$v$$$를 잇는 무방향 간선이라는 뜻이며, 주어지는 그래프는 트리이다. $$$N+1$$$번째 줄에 $$$N$$$개의 정수 $$$W_1, W_2, \cdots, W_N$$$이 공백으로 구분되어 주어진다. $$$(1\leq W_i \leq 10^9)$$$

Example

Input
5
2 3
3 1
4 1
1 5
1 2 2 3 4
Output
25
