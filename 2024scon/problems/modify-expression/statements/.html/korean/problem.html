수식 고치기
time limit per test
2 seconds
memory limit per test
1024 megabytes
input
standard input
output
standard output

T or F or T and F or T and F 와 같이 리터럴은 True 또는 False, 연산자는 or, and로만 구성된 수식이 있다. 연산자 or과 and의 연산자 우선순위가 같고, 왼쪽부터 오른쪽으로 차례로 계산한다고 가정한다. 여러분은 연산자 혹은 리터럴을 바꾸어 수식의 계산 결과가 참 혹은 거짓으로 계산되도록 해야 한다. 이때, 연산자와 리터럴을 바꾸는 횟수는 최소화해야 한다. 식과 원하는 계산 결과가 주어졌을 때, 원하는 계산 결과가 나오도록 리터럴 혹은 연산자를 고치는 최소 횟수를 구하여라.
