### 풀이

본 문제는 주어진 문자열 $S$와 $T$에 포함된 알파벳의 개수가 목표 문자열인 **"ICPC"**를 구성하기에 충분한지 판단하는 문제입니다.

1. 구성해야 할 문자열 "ICPC"의 문자 구성(`I`: 1개, `C`: 2개, `P`: 1개)을 기준치로 설정합니다.
2. 가방에 남은 문자열 $S$를 순회하며 해당 문자의 필요 개수를 줄입니다.
3. 스티커 문자열 $T$를 순회하며 추가로 보충 가능한 문자의 필요 개수를 줄입니다.
4. 마지막으로 문자열 "ICPC"를 이루는 각 문자의 필요 개수가 모두 $0$ 이하가 되었다면 `m4us happy`를, 하나라도 양수($>0$)로 남아있다면  `m4us sad`를 출력합니다.

---

### 시간 복잡도 분석
* $O(|S| + |T|)$  
  문자열 $S$와 $T$를 각각 한 번씩 순회하며 카운팅하므로, 문자열 길이에 비례하는 선형 시간 내에 해결 가능합니다. (1 \le $|S| \le 4, 1 \le |T| \le 100$)
