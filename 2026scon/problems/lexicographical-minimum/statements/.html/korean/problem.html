길이 $$$N$$$인 문자열 $$$T$$$가 주어진다. $$$T$$$는 알파벳 소문자로만 이루어져 있다.

당신은 다음 연산을 $$$0$$$회 이상 사용할 수 있다.

  • 두 정수 $$$l, r$$$를 선택한다. ($$$1 \le l \le r \le |T|$$$)
  • 만약 부분 문자열 $$$T_l T_{l+1} \dots T_r$$$의 모든 문자가 서로 같다면, 그 구간에서 문자 하나만 남기고 나머지를 모두 제거한다.

즉, $$$T_l = T_{l+1} = \dots = T_r$$$인 경우, 문자열 $$$ T_1 T_2 \dots T_{l-1}\, T_l T_{l+1} \dots T_r\, T_{r+1} \dots T_{|T|} $$$ 를 $$$ T_1 T_2 \dots T_{l-1}\, T_l\, T_{r+1} \dots T_{|T|} $$$ 로 바꿀 수 있다.

이 연산을 적절히 사용하여 만들 수 있는 모든 문자열 중, 사전 순으로 가장 앞서는 문자열을 구하여라.

Input

첫째 줄에 문자열의 길이 $$$N$$$이 주어진다. ($$$1 \le N \le 200\,000$$$)

둘째 줄에 길이 $$$N$$$인 문자열 $$$T$$$가 주어진다. $$$T$$$는 알파벳 소문자로만 이루어져 있다.

Output

가능한 모든 결과 문자열 중 사전 순으로 가장 앞서는 문자열을 출력한다.

Examples

Input
5
aabba
Output
aaba
Input
5
bbacc
Output
bac
