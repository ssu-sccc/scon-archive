\begin{problem}{덜 개성있는 구간}{standard input}{standard output}{1 second}{1024 megabytes}

다음 조건을 만족하는 길이가 $N(N \le 5000)$인 수열 $A = [A_1, A_2, \cdots, A_N](-10^{9} \le A_i \le 10^{9})$을 구하여라.

\begin{itemize}
    \item 수열 $A$의 모든 원소는 서로 다르다.
    \item $1 \le L \le R \le N$을 만족하는 모든 정수 쌍 $(L, R)$에 대하여, $\sum_{i=L}^R A_i$가 가질 수 있는 서로 다른 값의 개수는 정확히 N개이다.
\end{itemize}

주어지는 모든 입력에 대해 조건을 만족하는 수열은 항상 존재한다.

\InputFile
첫째 줄에 수열 $A$의 길이를 나타내는 정수 $N$이 주어진다.

\OutputFile
첫째 줄에 조건을 만족하는 수열 $A$의 원소들을 공백으로 구분하여 출력한다.

위 조건을 만족하는 수열이 여러 개라면 그중 아무거나 출력한다.

\Examples

\begin{example}
\exmpfile{example.01}{example.01.a}%
\exmpfile{example.02}{example.02.a}%
\end{example}

\Note
수열 $A = [-2, 0, 2]$에서 모든 연속 구간의 합을 구하면 다음과 같다.

\begin{itemize}
    \item $-2$: $(1, 1)$, $(1, 2)$
    \item $0$: $(1, 3)$, $(2, 2)$
    \item $2$: $(2, 3)$, $(3, 3)$
\end{itemize}

\end{problem}
