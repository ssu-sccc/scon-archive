구간 뒤집기

1 second

길이 $$$N$$$인 이진 문자열 $$$T$$$가 주어진다.

다음 연산을 원하는 만큼 사용할 수 있다.

  • 두 정수 $$$l, r$$$ ($$$1 \le l \le r \le N$$$)를 선택한다.
  • 부분 문자열 $$$T_l T_{l+1} \cdots T_r$$$의 모든 문자를 뒤집는다. 즉, 0은 1로, 1은 0으로 바뀐다.

문자열 $$$T$$$를 다음 조건을 만족하도록 바꾸고자 한다.

  • 모든 $$$1 \le i \lt N$$$에 대해 $$$T_i \ne T_{i+1}$$$

즉, 인접한 두 문자가 항상 서로 달라야 한다.

조건을 만족시키기 위해 필요한 연산의 최소 횟수를 구하여라.

Input

첫째 줄에 문자열의 길이 $$$N$$$이 주어진다. ($$$1 \le N \le 200\,000$$$)

둘째 줄에 길이 $$$N$$$인 이진 문자열 $$$T$$$가 주어진다.

Output

조건을 만족시키기 위해 필요한 연산의 최소 횟수를 출력한다.

Examples

Input
3
101
Output
0
Input
4
0011
Output
1
