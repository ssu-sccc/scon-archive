합의 최소
time limit per test
2 seconds
memory limit per test
256 megabytes
input
standard input
output
standard output

길이가 $$$N$$$인 정수로 구성된 수열 $$$A_1, A_2, ..., A_N$$$이 주어진다.

당신은 아래 연산을 원하는 만큼 0번 이상 사용하여 수열의 모든 원소들의 합 $$$\sum_{i=1}^N A_i$$$를 최소화하려고 한다.

  • $$$1$$$ 이상 $$$N-1$$$ 이하인 정수 $$$i$$$를 선택한 뒤, $$$A_i$$$의 값을 $$$A_{i+1}$$$로 변경한다.

만들 수 있는 수열의 합의 최솟값을 구하라.

Input

첫째 줄에 수열의 길이 $$$N(1 \leq N \leq 500\,000)$$$이 주어진다.

둘째 줄에 수열의 원소 $$$A_1, A_2, \cdots, A_N$$$이 공백으로 구분되어 주어진다. ($$$1 \le A_i \le 1\,000\,000$$$)

Output

수열의 합의 최솟값을 출력한다.

Examples

Input
5
5 4 3 2 1
Output
5
Input
5
4 2 7 3 6
Output
16
